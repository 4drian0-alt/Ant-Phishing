# 🛡️ Ant-Phishing

Projeto desenvolvido com HTML e CSS com o objetivo de conscientizar usuários sobre ataques de phishing e fornecer uma interface para análise de links suspeitos.

## 📌 Sobre o Projeto

O Ant-Phishing é uma aplicação web educativa que auxilia usuários na identificação de possíveis links maliciosos. Além da funcionalidade de verificação de URLs, o projeto oferece uma página educacional contendo boas práticas de segurança digital e prevenção contra golpes de phishing.

Este projeto foi desenvolvido como atividade acadêmica na graduação em Ciência da Computação.

---

## 🚀 Funcionalidades

### Página Inicial
- Inserção de links suspeitos para análise.
- Exibição do resultado da verificação.
- Histórico de links classificados como:
  - ✅ Seguros
  - ⚠️ Suspeitos

### Página Educacional
- Explicação sobre o que é phishing.
- Dicas de prevenção.
- Boas práticas de segurança digital.
- Link para monitoramento global de ataques cibernéticos.

## 🔒 O que é Phishing?

Phishing é uma técnica utilizada por criminosos para enganar usuários e obter informações confidenciais, como:

- Senhas
- Dados bancários
- Informações pessoais
- Códigos de autenticação

Normalmente os ataques ocorrem através de:

- E-mails falsos
- SMS fraudulentos
- Redes sociais
- Sites clonados
